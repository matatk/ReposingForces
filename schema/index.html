This is an API for Chess and variants such as Chess960. It is designed to be used by a client web site/app that actually presents UI and causes players to recieve an email when it's their turn to move.

The name is a pun on REST and Half-Life's 'Opposing Force' expansion pack.

Game

Stability: prototype

Represents a game of Chess. Note that a game may have more than one 'round' if the rules require it.

Attributes

Name Type Description Example
access_token string A randomly-generated (by the server) token that players should be given to enable them to access the game itself
pattern: ^[0-9a-z]{8}$		 "0d63vkm2"
created_at date-time When game was created "2012-01-01T12:00:00Z"
game_type string The game rules in use
one of:"chess" or "chess960" or "double-chess960"		 "double-chess960"
id integer Unique identifier of game 42
name string Name chosen for this game "Alice vs Bob: The Grudge"
player_one_email email Player one's email address "alice@chess.io"
player_one_name string Player one's name "Alice"
player_two_email email Player two's email address "bob@chess.io"
player_two_name string Player two's name "Bob"
updated_at date-time When game was updated "2012-01-01T12:00:42Z"

Game Create

Create a new game.

POST /games

Required Parameters

Name Type Description Example
game_type string The game rules in use
one of:"chess" or "chess960" or "double-chess960"		 "double-chess960"
name string Name chosen for this game "Alice vs Bob: The Grudge"
player_one_email email Player one's email address "alice@chess.io"
player_one_name string Player one's name "Alice"
player_two_email email Player two's email address "bob@chess.io"
player_two_name string Player two's name "Bob"

Curl Example

$ curl -n -X POST https://reposingforces.herokuapp.com/games \
  -d '{
  "name": "Alice vs Bob: The Grudge",
  "player_one_name": "Alice",
  "player_one_email": "alice@chess.io",
  "player_two_name": "Bob",
  "player_two_email": "bob@chess.io",
  "game_type": "double-chess960"
}' \
  -H "Content-Type: application/json"

Response Example

HTTP/1.1 201 Created
{
  "id": 42,
  "name": "Alice vs Bob: The Grudge",
  "game_type": "double-chess960",
  "player_one_name": "Alice",
  "player_one_email": "alice@chess.io",
  "player_two_name": "Bob",
  "player_two_email": "bob@chess.io",
  "created_at": "2012-01-01T12:00:00Z",
  "updated_at": "2012-01-01T12:00:42Z",
  "access_token": "0d63vkm2"
}

Game Public Info

Details of an existing game. If the access token is not given, then the players' email addresses are omitted.

GET /games/{game_id}

Curl Example

$ curl -n https://reposingforces.herokuapp.com/games/$GAME_ID

Response Example

HTTP/1.1 200 OK
{
  "id": 42,
  "name": "Alice vs Bob: The Grudge",
  "player_one_name": "Alice",
  "player_two_name": "Bob",
  "game_type": "double-chess960"
}

Game Full Info

Details of an existing game. If the access token is given, then all details are returned.

GET /games/{game_id}?access={game_access_token}

Curl Example

$ curl -n https://reposingforces.herokuapp.com/games/$GAME_ID?access=$GAME_ACCESS_TOKEN

Response Example

HTTP/1.1 200 OK
{
  "id": 42,
  "name": "Alice vs Bob: The Grudge",
  "player_one_name": "Alice",
  "player_one_email": "alice@chess.io",
  "player_two_name": "Bob",
  "player_two_email": "bob@chess.io",
  "game_type": "double-chess960"
}

Move

Stability: prototype

A move/the moves in a game.

Attributes

Name Type Description Example
move string The move "e4"

Move Add

Make a move.

POST /games/{game_id}/moves

Required Parameters

Name Type Description Example
access_token string A randomly-generated (by the server) token that players should be given to enable them to access the game itself
pattern: ^[0-9a-z]{8}$		 "0d63vkm2"
move string The move "e4"

Curl Example

$ curl -n -X POST https://reposingforces.herokuapp.com/games/$GAME_ID/moves \
  -d '{
  "move": "e4",
  "access_token": "0d63vkm2"
}' \
  -H "Content-Type: application/json"

Response Example

HTTP/1.1 201 Created
{
  "move": "e4"
}

Move List

List existing moves (the access token is not required).

GET /games/{game_id}/moves

Curl Example

$ curl -n https://reposingforces.herokuapp.com/games/$GAME_ID/moves

Response Example

HTTP/1.1 200 OK
[
  {
    "move": "e4"
  }
]
